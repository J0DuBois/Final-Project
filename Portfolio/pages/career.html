My path

Python

When I started to learn programming it was python that I used first. It was in 11th grade When I was introduced to it. As soon as I started and learned the simple functions I loved it. This love fueled my imagination, which caused me to create several projects. After I had completed thoes I then when on to learn other languages in order to create them there as well

C++

As I stated, I went to learn other languages in order to recreate the projects I had in python. That didn’t start until the following school year. It wasn't hard or difficult, but I felt like I wasn’t learning anything. This was because I was just copying what was in python to c++ but adapting its syntax. I didn't realize it until my first semester of college where the class was touching upon stuff, I already knew but, I was learning the language. And since then, I have loved working with both languages. .
